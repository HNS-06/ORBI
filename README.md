# StudyAI: AI Voice Study Companion (God Level)

A voice-first AI that studies with you, remembers you, and improves how you learn.

## 🚀 Vision
Not just another timer or chatbot. StudyAI is an **adaptive learning companion** that uses real-time voice interaction and cognitive memory to optimize student performance.

## 🔥 Core Features (Phases 1-4 Complete)

### 🎤 Voice-First Interaction
- **Neural Orb UI**: A reactive, pulsing visual center for all voice signals.
- **Continuous STT/TTS**: Real-time speech-to-text and text-to-speech powered by browser APIs and Gemini 2.0.
- **Confusion Detection**: Automatically detects hesitation or long pauses and offers empathetic help.

### 🧪 Intelligence Layer
- **Deep Recall Analysis**: AI analyzes your explanations for keywords, missing concepts, and clarity.
- **Memory System**: Your companion remembers your past performance and weak areas, referencing them in future sessions.
- **Adaptive Check-ins**: Smart AI nudges based on focus duration and historical performance.
- **Smart Break Logic**: Proactively suggests rest periods after 50 minutes of deep focus to prevent fatigue.

### 🎮 Addiction Engine
- **XP & Levels**: Earn rewards for focus time and recall mastery.
- **Achievements**: Unlock badges like "Recall Master" and "Deep Thinker".
- **Daily Streaks**: Encourages consistency with visual streak tracking.
- **Emotional AI**: Personality-consistent companions (Aria, Nexus, Vex) that build a bond with the user.

### 🛠️ Tech Stack
- **Frontend**: Vite, React, Tailwind CSS, Motion/React (Framer Motion).
- **AI**: Google Gemini 2.0 Flash (Dialogue, Recall Analysis, Check-ins).
- **Backend (Proposed)**: FastAPI (Initial endpoints implemented in `/backend`).
- **Storage**: LocalStorage (Phase 1-3), prepared for PostgreSQL/Redis integration.

## 🚀 Launch Strategy

### Step 1: MVP Testing
- Test with 10-20 active students.
- Monitor recall score improvements over 7 days.

### Step 2: Content Marketing
- Create short-form videos showing the "Neural Orb" in action.
- Highlight the "Explain like I'm five" feature.
- Catchphrase: "The app that actually listens to you study."

### Step 3: Monetization
- **Free**: 3 sessions/day, basic analysis.
- **Premium (₹299/mo)**: Unlimited AI sessions, premium voices, deep performance analytics, and custom personas.

## ⚠️ Developer Notes
- Ensure a valid `VITE_GEMINI_API_KEY` is set in `.env.local` or via the in-app Settings.
- Best experienced in Chrome for optimal Web Speech API support.
